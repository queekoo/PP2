# Moving Ball Game

An interactive game featuring a movable red ball, built with Pygame.

## Features
- Red ball (50×50 px, radius 25) on white background
- Move with arrow keys (Up, Down, Left, Right)
- Each key press moves the ball exactly 20 pixels
- Ball cannot leave the screen — off-screen input is ignored
- Displays current ball position on screen

## Setup
```bash
pip install pygame>=2.0.0
```

## Run
```bash
python main.py
```

## Controls
| Key        | Action         |
|------------|----------------|
| ↑ Up       | Move up        |
| ↓ Down     | Move down      |
| ← Left     | Move left      |
| → Right    | Move right     |
| ESC        | Quit           |
